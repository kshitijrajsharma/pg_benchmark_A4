## Setup Project 

We will use [UV](https://github.com/astral-sh/uv) to manage our dependencies 

- Install UV

```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```

- Install this project 
```bash
uv sync
``` 
- Run the notebook directly
